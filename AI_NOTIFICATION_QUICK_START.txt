═══════════════════════════════════════════════════════════════════════════════

DATA SOURCE:

- Endpoint: /api/user-{platform}-status/{userId}

🐛 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

NOTIFICATION NOT APPEARING:
1. Check console for errors
2. Verify backend is running (pm2 status)
3. Check network tab - are R2 status calls succeeding?
4. Clear sessionStorage: sessionStorage.removeItem('ai_notification_dismissed')
5. Hard refresh (Ctrl+Shift+R)

WRONG MESSAGE SHOWING:
1. Check backend R2 status endpoints
2. Verify platform acquisition in backend
3. Console log platform detection in component
4. Check userId is correct

STYLING ISSUES:
1. Check AIManagerNotification.css is imported
2. Clear browser cache
3. Check z-index (should be 2147483646)
4. Verify no CSS conflicts

CLICK NOT WORKING:
1. Check onOpenChat prop is passed correctly
2. Verify setIsOpen function in AIManagerChat
3. Check z-index stacking
4. Inspect for overlapping elements
